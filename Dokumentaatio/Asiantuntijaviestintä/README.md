# Tuote-esittely: Meteor-sovelluskehys

Meteor on Javascript-sovelluskehys. Se mahdollistaa käyttäjänpuolisen (front-end), sekä palvelinpuolisen (back-end) kehityksen. Palvelimen puolella 

Meteor-kehityksessä on todella helppo päästä alkuun. Meteorin kotisivuilla on helposti ymmärrettäviä ohjeita, joissa opastetaan aloittelija alkuun. Meteorilla on maailmanlaajuinen yhteisö, joka kehittää uusia kirjastoja, lisäosia ja web -sivujen ulkoasuja. Meteorissa kaikkia haastavimpia asioita ei edes tarvitse osata itse, vaan ohjelmistokehittäjä voi keskittyä enemmän ideoiden toteuttamiseen ja innovaatioiden kehittämiseen.



Sovelluskehitykset ja kehitysympäristöt voivat aina päätyä floppaamaan. Näin voi esimerkiksi käydä, jos sovelluskehystä kehittänyt yritys joutuu lopettamaan toimintansa. Jos ohjelmistokehitysopiskelijana alat opettelemaan Meteor-kehityksen perusteita, kokemus ei varmasti mene koskaan hukkaan. Meteorissa koodi kehitetään Javascript-ohjelmointikiellä. Kaikki tunnetuimmat internet-selaimet, kuten Chrome, Firefox ja Safari käyttävät Javascriptiä koodin ajamiseen käyttäjän selaimessa.
